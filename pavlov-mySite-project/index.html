О САЙТЕ

Данный сайт создан в целях демонстрации моих навыков для потенциальных работодателей и заказчиков.

Интерфейс выполнен по подобию PlayStation3. Возможно добавление новых разделов без изменения существующей структуры JS. Сам сайт адаптивен, быстро загружается и работает во всех современных браузерах.

Сайт создан на HTML, CSS и чистом JavaScript 2015 с использованием Gulp, Pug, SCSS и Babel. HTML и CSS прошли валидацию и соответствуют стандарту.
